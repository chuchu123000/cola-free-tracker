# Daniel老师不喝可乐打卡专用 🥤❌

一个可爱的每日打卡网页应用，帮助Daniel老师坚持不喝可乐的健康生活！

## ✨ 功能特点

- 📅 **日历打卡** - 直观的日历视图展示打卡历史
- 🔥 **连续统计** - 实时显示连续打卡天数
- 🏆 **成就系统** - 5个等级的成就徽章（新芽、小树、大树、勇士、王者）
- 👷 **可爱小人** - 戴白色安全帽的卡通形象陪伴你
- 💾 **数据持久化** - 使用localStorage保存数据，刷新不丢失
- 📱 **响应式设计** - 完美适配手机和电脑

## 🎨 界面预览

- 渐变紫色背景，温馨可爱
- 可爱的戴安全帽小人形象
- 打卡成功彩纸庆祝动画
- 成就徽章解锁系统

## 🚀 在线访问

访问地址：[https://zhangchu05.github.io/cola-free-tracker/](https://zhangchu05.github.io/cola-free-tracker/)

## 💪 使用方法

1. 点击"今日打卡"按钮记录每天没喝可乐
2. 查看日历了解打卡历史
3. 跟踪连续打卡天数和总打卡数
4. 解锁不同等级的成就徽章

## 🏅 成就等级

- 🌱 新芽 - 连续3天
- 🌿 小树 - 连续7天
- 🌳 大树 - 连续14天
- 🎖️ 勇士 - 连续30天
- 👑 王者 - 连续100天

## 🛠️ 本地运行

```bash
# 克隆仓库
git clone https://github.com/zhangchu05/cola-free-tracker.git

# 进入目录
cd cola-free-tracker

# 启动本地服务器（任选一种）
python3 -m http.server 8080
# 或
npx serve

# 在浏览器打开
# http://localhost:8080
```
